# ProyectoRedes
# instalacion del proyecto
1. Creacion del entorno virtual:
```bash
python -m venv venv
```
2. activacion del entorno virtual:
```bash
# En Linux
source venv/bin/activate
```

3. instalacion de las dependencias:
```bash
pip install -r requirements.txt
```
